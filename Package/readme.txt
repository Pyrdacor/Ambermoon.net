Thank you for playing Ambermoon.net. If you have any questions, suggestions or other
feedback, feel free to visit the project on github: https://github.com/Pyrdacor/Ambermoon.net.



            ------------------------------------------------------------------
            |                              Content                           |
            ------------------------------------------------------------------


This package includes the playable assembly for Ambermoon.net and some additional resources.

- Walkthrough.txt: An english walkthrough written by Michael Böhnisch in 1999 and kindly
                   provided by Alexander Holland from the Thalion Webshrine.
- AmbermoonMap.pdf: A scan of the Ambermoon map that was part of the original package.
- AmbermoonRuneTable.png: A scan of the rune table that was part of the original package.
- AmbermoonManualEnglish.pdf: A translated manual by Victor Chaves (format edited by me).

Note: The manual is meant for the original so especially for the controls or Amiga stuff
      there might be differences in Ambermoon.net. If you want the original german manual
      in high quality you can find it here: https://tinyurl.com/AmbermoonManual
