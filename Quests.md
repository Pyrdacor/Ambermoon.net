# Ambermoon Quests

- Main quests with name
- Subquests with description (steps)
- Dependency of main quests (e.g. finish Tolimar quest to continue grandfather's quest)
- Source (person name)
- Amount updates (like find 1/3 items and later 2/3, 3/3)
